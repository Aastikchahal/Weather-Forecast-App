# Weather Dashboard

This is a simple web application that allows users to search for weather information by entering a city name or using their current location. It also provides a dropdown menu to quickly access recently searched cities.

## Features

- Search for weather information by city name
- Use the current location to get weather data
- View current weather details (temperature, wind, humidity) for the searched city
- See a 5-day forecast for the searched city
- Recently searched cities are saved and displayed in a dropdown menu
- Clicking on a city in the dropdown menu updates the weather information

## Setup Instructions

1. Clone the repository or download the source code.
2. Open the project in your preferred code editor.
3. Replace the `API_KEY` variable in the JavaScript file with your OpenWeatherMap API key. You can obtain an API key by creating an account on [OpenWeatherMap](https://openweathermap.org/).
4. Open the `index.html` file in a web browser.

## Usage

1. Enter a city name in the input field and click the "Search" button, or press the "Enter" key.
2. Alternatively, click the "Use Current Location" button to get weather information for your current location (requires location permission).
3. The current weather details and a 5-day forecast for the searched city will be displayed.
4. Recently searched cities will be saved and displayed in the dropdown menu below the search input.
5. To view weather information for a recently searched city, select it from the dropdown menu.

## Dependencies

This project uses the following third-party API:

- [OpenWeatherMap](https://openweathermap.org/api)
